**Firefox icon asset:**
Download or create a simple Firefox logo PNG (48x48 or 64x64) and save to `raycast-extension/assets/firefox-icon.png`. This is used as fallback icon for about:* internal pages. A simple approach: use a small orange/red circle or download the Firefox logo from a public CDN.

**Update the Tab interface** to include the new fields:
- Add `lastAccessed: number` (timestamp from Firefox)
- Update `container` type to include `colorCode: string` alongside existing fields:
  `container: { name: string; color: string; colorCode: string; icon: string } | null`

**Add helper functions** (place above the component):

1. `cleanUrl(url: string): string` -- Extract hostname (remove www.) + pathname (omit if just "/"). Returns cleaned display string. For invalid URLs, return the raw URL.

2. `getHostname(url: string): string` -- Extract hostname from URL. Return empty string on parse failure.

3. `domainColor(hostname: string): string` -- djb2 hash of hostname to select from a 14-color palette. Use the same palette as @raycast/utils getAvatarIcon:
   `["#DC829A","#D64854","#D47600","#D36CDD","#52A9E4","#7871E8","#70920F","#43B93A","#EB6B3E","#26B795","#D85A9B","#A067DC","#BD9500","#5385D9"]`

4. `getFallbackIcon(tab: Tab): Image.ImageLike` -- Returns appropriate fallback icon:
   - `tab.url.startsWith("about:")` -> `{ source: "firefox-icon.png" }`
   - `tab.status === "loading"` -> `Icon.CircleProgress`
   - Otherwise -> `getAvatarIcon(letter, { background: domainColor(hostname) })` where letter is first char of title (or hostname, or "?")

5. `getWindowNumber(windowId: number, allWindowIds: number[]): number` -- Return 1-based index of windowId in sorted unique windowId array.

6. `buildAccessories(tab: Tab, windowNumber: number): List.Item.Accessory[]` -- Build accessories array in this order:
   - If `tab.pinned`: `{ icon: Icon.Pin, tooltip: "Pinned" }`
   - If `tab.active`: `{ tag: { value: "Active", color: Color.Green } }`
   - If `tab.container`: `{ tag: { value: tab.container.name, color: tab.container.colorCode || Color.SecondaryText } }`
   - Always: `{ tag: { value: "W" + windowNumber, color: Color.SecondaryText } }`

**Update urlKeywords function:**
Add the full URL string as the first keyword so search still matches protocol and query params even though subtitle now shows cleaned URL:

return [url, hostname, ...hostParts, ...pathParts];

2. Compute unique window IDs (sorted) for window number assignment:
   ```typescript
   const windowIds = [...new Set(sortedTabs.map(t => t.windowId))].sort((a, b) => a - b);
   ```

   - `accessories`: `buildAccessories(tab, getWindowNumber(tab.windowId, windowIds))`
   - `actions`: Keep existing ActionPanel unchanged
