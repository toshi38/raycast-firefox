    Create the `raycast-extension/metadata/` directory. Create 3 tiny placeholder PNG files with the correct naming convention:

    - `firefox-tabs-3.png` (Setup toast flow)

    These are placeholders -- the user will replace them with actual 2000x1250 screenshots captured via Raycast's window capture (Cmd+Shift+Alt+M). Create minimal valid PNGs (even 1x1 pixel) so the folder structure exists. Add a `metadata/README.md` is NOT needed -- just the PNG files.

    To create minimal placeholder PNGs, use sips to create a small image:
    ```bash
    mkdir -p raycast-extension/metadata
    # Create a 1x1 placeholder PNG using sips
    sips -z 1 1 raycast-extension/assets/icon.png --out /tmp/placeholder.png 2>/dev/null
    cp /tmp/placeholder.png raycast-extension/metadata/firefox-tabs-1.png
    cp /tmp/placeholder.png raycast-extension/metadata/firefox-tabs-2.png
    cp /tmp/placeholder.png raycast-extension/metadata/firefox-tabs-3.png
    ```

    **5. Run `ray build` to verify compilation:**
    Run `cd raycast-extension && npx ray build -e dist` to verify the extension compiles cleanly with all the changes.

    **6. Run final `ray lint` to verify zero errors:**
    Run `cd raycast-extension && npx ray lint` and confirm zero errors. If any remain, fix them.
