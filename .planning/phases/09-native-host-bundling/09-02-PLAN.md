  <files>native-host/run.sh</files>
  <action>
Rewrite `native-host/run.sh` with the following structure:

```bash
#!/bin/bash
set -euo pipefail

DIR="$(cd "$(dirname "$0")" && pwd)"
BUNDLE="$DIR/host.bundle.js"
LOG_DIR="$HOME/.raycast-firefox/logs"
MIN_NODE_VERSION=18

mkdir -p "$LOG_DIR"
```

1. **Dual entry point detection**: If `host.bundle.js` exists in `$DIR`, use it as `$BUNDLE`. Otherwise fall back to `$DIR/host.js` (dev mode — running from source checkout). This allows the same `run.sh` to work in both dev (alongside `host.js`) and production (alongside `host.bundle.js`).

2. **Logging functions**: `log_info()` and `log_and_exit()` that write timestamped messages to `$LOG_DIR/wrapper.log` (ISO 8601 UTC timestamps).

3. **Version checking function** `check_node_version()`:
   - Run `"$node_path" --version` to get version string
   - Extract major version: `echo "$version" | sed 's/^v//' | cut -d. -f1`
   - Return 0 if major >= `$MIN_NODE_VERSION`, else log skip message and return 1

4. **Try function** `try_node()` that takes `(node_path, label)`:
   - Check `[ -x "$node_path" ]` and `check_node_version "$node_path"`
   - On success: get version, call `log_info "Using $label: $node_path ($version)"`, then `exec "$node_path" "$BUNDLE"`

5. **Priority chain** (exact order per user decision):
   - Priority 1: Raycast bundled — `find "$HOME/Library/Application Support/com.raycast.macos/NodeJS/runtime" -name "node" -type f 2>/dev/null | head -1`
   - Priority 2: Homebrew ARM — `/opt/homebrew/bin/node`
   - Priority 3: Homebrew Intel — `/usr/local/bin/node`
   - Priority 4: nvm — `ls -d "$HOME/.nvm/versions/node"/v*/bin/node 2>/dev/null | sort -V | tail -1` (latest version)
   - Priority 5: system PATH — `command -v node`

6. **Failure case**: Call `log_and_exit` with a message listing all paths checked, exit non-zero.

Important: The wrapper must NOT use `"$@"` after `$BUNDLE` in the exec — the native messaging host receives data on stdin, not as arguments. The `"$@"` was harmless before but should be removed for clarity.

Implementation:
```typescript
const PRODUCTION_RUN_SH = join(
  homedir(),
  ".raycast-firefox",
  "bin",
  "run.sh",
);

export function resolveNativeHostPath(): string {
  // Dev path: project-root.txt exists -> use repo checkout
  const rootFile = join(environment.assetsPath, "project-root.txt");
  if (existsSync(rootFile)) {
    const projectRoot = readFileSync(rootFile, "utf-8").trim();
    const devPath = join(projectRoot, "native-host", "run.sh");
    if (existsSync(devPath)) {
      return resolve(devPath);
    }
  }

  // Production path: installed bundle
  if (existsSync(PRODUCTION_RUN_SH)) {
    return resolve(PRODUCTION_RUN_SH);
  }

  // Neither found
  throw new Error(
    "Native host not found. Run the 'Setup Firefox Bridge' command to install it.",
  );
}
```
