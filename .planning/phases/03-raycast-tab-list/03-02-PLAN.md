<task type="checkpoint:human-verify" gate="blocking">
  <what-built>
    Search Firefox Tabs command: A Raycast command that fetches all open Firefox tabs from the native messaging bridge and displays them as a searchable list with title and URL.
  </what-built>
  <how-to-verify>
    1. Open Raycast (Cmd+Space or your hotkey)
    2. Type "Search Firefox Tabs" and press Enter
    3. Verify: A list of your open Firefox tabs appears with titles and URLs
    4. Verify: Each tab shows the page title as the main text and the URL below it
    5. Type part of a tab's title (e.g., "github") -- verify the list filters to matching tabs
    6. Clear search, type part of a URL domain (e.g., "google") -- verify tabs with that domain appear
    7. Close and re-open the command -- verify the list reflects current Firefox tabs (open/close a tab in Firefox to confirm)

    Expected:
    - Tab titles display correctly as primary text
    - URLs display as subtitle text below each title
    - Fuzzy search works on both title text and URL text
    - Re-invoking the command fetches fresh data

    If tabs do NOT appear:
    - Check Firefox is running with the WebExtension installed
    - Check `curl http://127.0.0.1:26394/tabs` returns tab data
    - Check `cat ~/.raycast-firefox/port` matches the port the host is using
  </how-to-verify>
  <resume-signal>Type "approved" if all checks pass, or describe any issues.</resume-signal>
</task>
